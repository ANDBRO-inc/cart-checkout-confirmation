Cart Checkout Confirmation plugin will give you a step to confirm information to proceed to checkout

== Description ==

Cart Checkout Confirmation is a plugin that allows you to insert a confirmation page between WooCommerce's purchase screen (enter customer information) and the checkout completion screen.

== Installation ==

1. Cart Checkout Confirmation can be installed directly through your WordPress
Plugins dashboard.
1. Click "Add New" and Search for "Cart Checkout Confirm"
1. Install and Activate
